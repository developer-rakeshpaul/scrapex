Parsers extract structured data from Markdown content.

## MarkdownParser

Parses Markdown content to extract headings and links.

```typescript
class MarkdownParser {
  parse(markdown: string): MarkdownParseResult;
}
```

### MarkdownParseResult

```typescript
interface MarkdownParseResult {
  headings: Heading[];
  links: MarkdownLink[];
}

interface Heading {
  level: number;  // 1-6
  text: string;
}

interface MarkdownLink {
  url: string;
  text: string;
  description: string;
}
```

### Example

```typescript
import { MarkdownParser } from 'scrapex/parsers';

const parser = new MarkdownParser();

const result = parser.parse(`
# Title

Some text with a [link](https://example.com).

## Section

- [Item 1](https://one.com) - First item
- [Item 2](https://two.com) - Second item
`);

console.log(result.headings);
// [
//   { level: 1, text: 'Title' },
//   { level: 2, text: 'Section' },
// ]

console.log(result.links);
// [
//   { url: 'https://example.com', text: 'link', description: '' },
//   { url: 'https://one.com', text: 'Item 1', description: 'First item' },
//   { url: 'https://two.com', text: 'Item 2', description: 'Second item' },
// ]
```

### Link Description Patterns

The parser extracts descriptions from these patterns:

```markdown
- [Link](url) - Description with dash
- [Link](url): Description with colon
- [Link](url) â€” Description with em dash
```

## parseAwesomeList()

Specialized parser for GitHub awesome lists.

```typescript
function parseAwesomeList(markdown: string): AwesomeList
```

### AwesomeList

```typescript
interface AwesomeList {
  title: string;
  description: string;
  sections: AwesomeSection[];
}

interface AwesomeSection {
  name: string;
  level: number;
  items: AwesomeItem[];
  subsections?: AwesomeSection[];
}

interface AwesomeItem {
  name: string;
  url: string;
  description: string;
}
```

### Example

```typescript
import { parseAwesomeList } from 'scrapex/parsers';

const markdown = `
# Awesome Tools

A curated list of awesome tools.

## Editors

- [VS Code](https://code.visualstudio.com) - Free and open source
- [Sublime Text](https://sublimetext.com) - Sophisticated text editor

### Vim-based

- [Neovim](https://neovim.io) - Hyperextensible Vim-based editor

## Terminals

- [iTerm2](https://iterm2.com) - macOS terminal replacement
- [Warp](https://warp.dev) - Modern terminal with AI
`;

const result = parseAwesomeList(markdown);

console.log(result.title);
// 'Awesome Tools'

console.log(result.description);
// 'A curated list of awesome tools.'

console.log(result.sections[0]);
// {
//   name: 'Editors',
//   level: 2,
//   items: [
//     { name: 'VS Code', url: 'https://code.visualstudio.com', description: 'Free and open source' },
//     { name: 'Sublime Text', url: 'https://sublimetext.com', description: 'Sophisticated text editor' },
//   ],
//   subsections: [
//     {
//       name: 'Vim-based',
//       level: 3,
//       items: [
//         { name: 'Neovim', url: 'https://neovim.io', description: 'Hyperextensible Vim-based editor' },
//       ],
//     },
//   ],
// }
```

## GithubMarkdownParser

Extended parser for GitHub-flavored Markdown.

```typescript
class GithubMarkdownParser extends MarkdownParser {
  parseGithubReadme(markdown: string): GithubReadmeResult;
}
```

### GithubReadmeResult

```typescript
interface GithubReadmeResult extends MarkdownParseResult {
  badges: Badge[];
  tableOfContents: TOCEntry[];
}

interface Badge {
  alt: string;
  url: string;
  link?: string;
}

interface TOCEntry {
  text: string;
  anchor: string;
  level: number;
}
```

### Example

```typescript
import { GithubMarkdownParser } from 'scrapex/parsers';

const parser = new GithubMarkdownParser();

const result = parser.parseGithubReadme(`
# My Project

[![npm](https://img.shields.io/npm/v/my-project)](https://npmjs.com/package/my-project)
[![CI](https://github.com/user/repo/actions/workflows/ci.yml/badge.svg)](https://github.com/user/repo/actions)

## Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

\`\`\`bash
npm install my-project
\`\`\`
`);

console.log(result.badges);
// [
//   { alt: 'npm', url: 'https://img.shields.io/npm/v/my-project', link: 'https://npmjs.com/package/my-project' },
//   { alt: 'CI', url: '...badge.svg', link: '...actions' },
// ]

console.log(result.tableOfContents);
// [
//   { text: 'Installation', anchor: 'installation', level: 2 },
//   { text: 'Usage', anchor: 'usage', level: 2 },
// ]
```

## Utilities

### Flattening Sections

Get all items from an awesome list:

```typescript
function flattenAwesomeList(list: AwesomeList): AwesomeItem[] {
  const items: AwesomeItem[] = [];

  function processSection(section: AwesomeSection) {
    items.push(...section.items);
    section.subsections?.forEach(processSection);
  }

  list.sections.forEach(processSection);
  return items;
}

const allItems = flattenAwesomeList(result);
```

### Filtering by Section

```typescript
const section = result.sections.find(s => s.name === 'Editors');
const editorLinks = section?.items ?? [];
```
